# Spring boot

## Project Description
It is consist of multiple sub projects where each having Crud operation, where JWT authentication need to be implemented shortly.
 
## Sub Projects
1. College API
2. Banking API
3. CRUD with MYSQL
4. Registration


## Project Tech Stack
- Language: Java
- Components/Lib: DataJPA, Spring web, mySql connector, actuator.
- Hosting: localhost
- Version Control: Git, GitHub

## Project Architecture
This all the projects follows MVC architecture for the api creation.

## Future Implementation
Authentication with JWT token implementation.
Google and Apple Login implementaion with our API.
